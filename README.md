# moltbot-control-ui-custom

A respectful, minimal customization layer for the **Moltbot/Clawdbot Control UI**.

## What
- Adds a right-side **Activity (terminal-like) panel** so long-running runs don't look like a freeze.
- Auto-follow logs; user scroll locks; includes both wall-clock and elapsed timestamps.
